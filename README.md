# Chennai Metro Interactive Map
An interactive transportation map of Chennai that visualizes CMRL metro, Indian railways, and future transport integrations.

## Developing
```sh
# Install dependancies 
npm install
```

```sh
npm run dev
# or start the server and open the app in a new browser tab
npm run dev -- --open
```
## Building
To create a production version of your app:
```sh
npm run build
```

You can preview the production build with `npm run preview`.
